# ⚡️ CYRAX SHOP & BOT

Премиальный веб-магазин и Telegram бот для продажи ключей и услуг. Единая экосистема с полной синхронизацией данных через SQLite.

## 🚀 Основные возможности

### 🤖 Telegram Бот
- Автоматическая продажа ключей 24/7.
- Поддержка CryptoBot (автоматически) и ручных методов оплаты (СБП, Карта, Binance).
- Система ресселеров и партнерская программа.
- Уведомления администратора о каждом заказе.

### 🌐 Веб-сайт (Storefront)
- **Премиальный дизайн**: Глубокий темный режим с неоновыми акцентами в стиле luxury.
- **Обязательная авторизация**: Вход только через Telegram Widget (защита от ботов и накруток).
- **Синхронизация баланса**: Общий кошелек между сайтом и ботом.
- **Поддержка онлайн**: Виджет чата на сайте с проксированием сообщений в Telegram админа.
- **Админ-панель**: Управление заказами прямо из браузера.
- **Maintenance Mode**: Возможность мгновенно перевести сайт в режим техобслуживания.

## 🛠 Технологический стек
- **Backend**: Node.js + Express (API прокси).
- **Database**: SQLite (общая база `shop.db`).
- **Frontend**: Vanilla JS + CSS3 (Glassmorphism).
- **Hosting**: Рекомендуется Render (Backend) + Vercel (Frontend).

## 📦 Установка и запуск

### 1. Подготовка окружения
Создайте файл `.env` в корневом каталоге (на базе приложенных примеров):
```env
BOT_TOKEN=ваш_токен
ADMIN_ID=ваш_id
ADMIN_USERNAME=ваш_ник
CRYPTOBOT_TOKEN=токен_криптобота
WEBHOOK_SECRET=ваш_секрет_32_символа
API_BASE=https://ваш-бэкенд.onrender.com/api
```

### 2. Запуск бэкенда
```bash
npm install
node index.js
```

### 3. Развертывание фронтенда
Загрузите содержимое папки `site/` на Vercel или любой статический хостинг. Убедитесь, что в `script.js` указан правильный `API_BASE`.

## 🔐 Безопасность
- **Приватный репозиторий**: Настоятельно рекомендуется сделать репозиторий **ПРИВАТНЫМ**, так как он содержит бизнес-логику и настройки безопасности.
- **Webhook Secret**: Никогда не делитесь своим `WEBHOOK_SECRET`.
- **Admin ID**: Доступ к админ-панели жестко привязан к `ADMIN_ID` в базе данных.
